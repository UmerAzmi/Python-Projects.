# To-Do List Application

A simple console-based To-Do List application built using Python.
The program allows users to add, view, and remove tasks through a
menu-driven interface.

This project was created to practice working with lists, functions,
loops, and basic input validation.

## Features
- Add new tasks
- View all existing tasks with numbering
- Remove tasks by selecting task number
- Continuous menu until user exits
- Basic error handling for invalid input

## Concepts Used
- Lists
- Functions
- while loops
- enumerate()
- Conditional statements
- try / except for input validation

## How to Run
```bash
python todo_list.py
