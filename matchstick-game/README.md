# Matchstick Game

A console-based Matchstick Game built using Python.
The game starts with 21 matchsticks, and the player competes against
the computer by picking 1 to 4 matchsticks per turn.

The player who picks the last matchstick loses.

## Game Rules
- The game starts with 21 matchsticks
- Players can pick 1 to 4 matchsticks per turn
- The computer plays optimally
- Picking the last matchstick results in a loss
- Enter `0` to exit the game at any time

## Features
- User vs computer gameplay
- Input validation for invalid and out-of-range values
- Optimal computer strategy
- Graceful exit option
- Turn-based game flow

## Concepts Used
- while loops
- Conditional statements
- Input validation
- Exception handling (`try / except`)
- Game logic and strategy
