# Adventure into the Forgotten Forest

A text-based adventure game where the player explores a mysterious world
and makes choices that affect the story and ending.

The player selects a character name and a special power, which influences
available paths and outcomes.

## Features
- Interactive story with multiple paths
- Player-selected powers (Strength, Magic, Stealth)
- Branching decisions (forest, cave, village, ruins, mountains)
- Multiple endings based on choices

## Concepts Used
- Functions
- Conditionals
- User input handling
- Game flow control
- Story branching logic
