# Random Number Guesser

A simple Python game where the user tries to guess a randomly generated
number until they get it correct.

The program provides feedback after each guess and keeps track of the
number of attempts.

## Features
- Random number generation
- High/low hints after each guess
- Input validation
- Attempt counter

## Concepts Used
- while loops
- if / else logic
- random module
- User input validation

## How to Run
```bash
python random_number_guesser.py
